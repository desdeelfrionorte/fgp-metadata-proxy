Description


  • This custom transformer is translating the input CKAN metadata model into the JSON TBS model. The rules on how to map the input CKAN features attributes into the TBS attributes are stored in the JSON configuration file. The JSON templates used to create the JSON file are supplied with the input parameters of the custom transformers. The mapping into JSON file is done with FME JSON Templater transformer.



Input Ports


  • Input:The input metadata records to translate into the JSON TBS model.

  • Config file:The content of the JSON config file for the province to process.



Output Ports


  • OUTPUT_LOCAL: The output metdata records translated into the JSON TBS model.



Parameters


  • JSON_ROOT_TEMPLATE: File system address containing the root section of the TBS model in JSON format.

  • JSON_RESOURCE_TEMPLATE:File system address containing the resources section of the TBS model in JSON forma.
