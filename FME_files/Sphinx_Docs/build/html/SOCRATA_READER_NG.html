SOCRATA_READER_NG

Documentation du Custom Transformer FME

Description

This Custom Transformer read the content of a Socrata server. All the information is read and all the information is transformed into FME attributes.

Input Ports

  • Input: One feature needed to start the reading of the Socrata server.

Output Ports

  • Output: One feature for each metadata record read from the Socrata server.

Parameters

  • Domain: The url of the Socrata server.

Documentation du code SOCRATA_READER_NG.py

SOCRATA_READER_NG

class SOCRATA_READER_NG.ManageHttpCall[source]

Bases: object

This class enables the call to Socrata GeoHub servers and the creation of FME features with Socrata information.

To extract all the information the Socrata reader makes a first http call to extract all the IDs then a second http call is done to extract the information of each ID. The information is finally stored in FME attributes.
