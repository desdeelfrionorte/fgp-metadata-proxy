FORMAT_VALIDATOR

Documentation du Custom Transformer FME

Description

This custom transformer validates and corrects resources when the format is WMS. when the format is WMS the URL must contained the following substring "service=wms" otherwise the format is set to "other".

Input Ports

  • INPUT: The input metadata records to process.

Output Ports

  • OUTPUT: The process output metdata records with the format validated.

Parameters

NONE
