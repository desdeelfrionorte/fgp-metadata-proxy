Description


  • This custom transformer sets manually the spatial representation type. The supplied lookup table contains the ID and the spatial representation type for a subset of the metadata records.



Input Ports


  • LOOKUP_TABLE_INPUT: Input information from the lookup table


  • DATA_INPUT: The input metadata records to process.



Output Ports


  • GEOSPATIAL: Output port for the geospatial metadata records


  • NON-GEOSPATIAL: Output port for the non geospatial metadata records


  • UNMAPPED_DATA: Output port for the metadata records for which the spatial representation type cannot be determine.



Parameters


  • None
