Description


  • This custom transformer attributes a spatial representation type to the input metadata records. There are three representation spatial types: grid, vector and textTable. In order to determine the spatial type, the custom transformer uses 2 distinct strategies: firstly it analyses the resources{}.spatial_type to extract the spatial type; secondly, if the first strategy fails, the custom transformer is using a lookup table containing the spatial representation type for a list of specific ID (uuid).



Input Ports


  • LOOKUP_TABLE_INPUT: Input information from the lookup table


  • DATA_INPUT: The input metadata records to process.



Output Ports


  • DATA_OUTPUT_PASSED: The output metdata records for which the spatial representation type was successfully set.


  • DATA_OUTPUT_FAILED: The output metdata records for which the spatial representation type was not successfully set.



Parameters


  • No parameter needed
