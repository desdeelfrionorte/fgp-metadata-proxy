Description


  • This custom transformer validates the URL, the FORMAT and the NAME attributes of each list of the resources{} of the metadata records. If the URL is missing this list is dropped. If the FORMAT is missing the value OTHER is added. If the format is missing the value of the TITLE attribute is supplied.



Input Ports


  • INPUT: Input metadata records toprocess



Output Ports


  • OUTPUT: The process output metdata records with resources validated



Parameters


  • NONE
