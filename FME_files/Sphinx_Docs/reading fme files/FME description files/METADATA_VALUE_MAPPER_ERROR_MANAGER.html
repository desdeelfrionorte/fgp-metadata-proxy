Description


  • This custom transformer checks if the input records have mapping errors. A record has one or more mapping errors if the attribute mapping_errors{}. is present. If mapping errors are present the individual errors are individually extracted from the list (mapping_errors{} list is exploded) and sent to the ERROR_OUTPUT port.


  • Mapping errors are created by the following custom transformers: METADATA_VALUE_MAPPER, METADATA_FORMAT_MAPPER



Input Ports


  • Input: The input metadata records to process.



Output Ports


  • ERROR_OUTPUT: The detailed list of errors (one record per error)
  • DATA_OUTPUT: The metadata record without any mapping error



Parameters


  • None
