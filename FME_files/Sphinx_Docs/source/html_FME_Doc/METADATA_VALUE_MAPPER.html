Description

This custom transformer corrects (maps) values to valid English values and valid French values equivalent and RI_CODES. The name of the attribute to map and French, English and RI_CODES action are parametrizable.

Input Ports

LOOKUP_TABLE_INPUT: Input information from the lookup table

DATA_INPUT: The input metadata records to process.

Output Ports

OUTPUT: The process output metdata records with value mapped.

Parameters

  • ATT_TO_MAP: Name of the FME attribute to map in plain text (ex.: resources{}.format). Do not use the Open Text Editor > FME Feature Attribute option to select the attribute to map

  • ERROR_NOT_MAPPED: Flag use to map data items values that cannot be found in the loopup tables. Can take 2 values: Yes: map data values not found; No: Do not map data values not found

  • FRENCH_REFRESH: Flag use for updating French attribute value. Can take two value: Yes: update French attribute value; No do not update French atribute value

  • ENGLISH_REFRESH: Flag use for updating English attribute value. Can take two value: Yes: update English attribute value; No do not update English attribute value

  • CODE_REFRESH: Flag use for updating the RI_CODES. Can take two value: Yes: update the code; No do not update the code
