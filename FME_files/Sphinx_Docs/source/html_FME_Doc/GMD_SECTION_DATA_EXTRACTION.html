Description

This transformer copies original lists then removes duplicates from elements where only unique values are required for the dataset. Duplicate attribute format are removed. Duplicate projection name are removed. Only the first occurrence of the cycle and cycle_code attributes are kept. Duplicate more_info{}.link are removed

Input Ports

  • INPUT: Input metadata records to process.

Output Ports

  • DATA_OUTPUT: The process output metdata records.

Parameters

  • NONE
