Description

This custom transformer allows to sort the metadata records based on the type of license. If the licence allows to publish the metadata records in the FGP, the metadata records are sent to the LICENSE_FILTER_PASSED port.

Input Ports

  • LOOKUP_TABLE_INPUT: Input information from the lookup table about the different type of licence.

  • DATA_INPUT: The input metadata records to process.

Output Ports

  • LICENSE_FILTER_PASSED: The metadata records that have a license allowing to be publish in the FGP.

  • UNKNOWN_LICENCE: The metadata records with a license not found in the lookup table.

Parameters

  • LICENCE_ATT: Name of the attribute of the metadata record containing the licence type used to determine if the metadata records can be published in the FGP.
