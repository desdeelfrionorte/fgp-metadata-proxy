Description

This custom transformer will add spatial extents to metadata records that do not have spatial extents. For the matadata records without a spatial extent, the spatial extents are extracted from a lookup table based on the location attribute of the metadata record. If the spatial extent is not set and the location is not present in the lookup table the spatial extent cannot be set and a description is added in the log file.

Input Ports

  • LOOKUP_TABLE_INPUT: Input extents from the lookup table.

  • INPUT: The input metadata records to process.

Output Ports

  • EXTENT_DEFINED: The output metdata records for which the spatial extent is defined (already present and/or added).

  • UNKNOWN_EXTENT: The output metdata records for which the spatial extent is unknown (not defined).

Parameters

  • LATITUDE_MIN: Minimum latitude for all the entries of the lookup table.

  • LATITUDE_MAX: Maximum latitude for all the entries of the lookup table.

  • LONGITUDE_MIN: Minimum longitude for all the entries of the lookup table.

  • LONGITUDE_MAX: Maximum longitude for all the entries of the lookup table.
