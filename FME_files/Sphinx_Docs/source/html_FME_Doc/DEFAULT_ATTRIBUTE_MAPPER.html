Description

This custom transformer maps each value pair (key, value) from the lookup table as attribute into each metadata record. If there are 25 value pairs in the lookup table each metadata records will have 25 new attributes.

Input Ports

LOOKUP_TABLE_INPUT: Input value pair information (key, value) from the lookup table

DATA_INPUT: The input metadata records to process.

Output Ports OUTPUT: The process output metdata records with new attributes (key, value).

Parameters

None
