Description

This custom transformer is doing 2 tasks. Firstly, it creates and validates that the web link is well formed, online and responding; Secondly it is adding this web link at the end of the resources{} list of the metadata record. Unvalid web links are log in the FME logger.

Input Ports

  • LOOKUP_TABLE_INPUT: Input information from the lookup table to build the additional resource information.

  • DATA_INPUT: The input metadata records to process.

Output Ports

  • OUTPUT: The process output metdata records with the added web link resource.

Parameters

  • WEBLINK_WILDCARD: Name of the attribute containing the information to add to the web link in order to retreive the metadata record from the portal of the province.

  • URL_VALIDATION: Flag use to validate if the web link is well formed, online and responding. Can take 2 values: YES: validate the web link; NO: Do not validate th web link. Note: During testing, it is interesting to use NO as the validation is slow.
