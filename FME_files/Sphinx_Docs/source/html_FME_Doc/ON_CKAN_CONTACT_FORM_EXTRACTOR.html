Description:

Custom transformer désigné pour l'extraction des formulaires de contact pour pour l'ontario. Pour l'Ontario, chaque organisation a sa propre page de formulaire pour permettre aux usagers de les contater

Dépendance:

Ce custom se connecte à l'API CKAN pour extraire org_contact.fr et org_contact.en.

Input Ports

Input: Métadonnées de la province de l'Ontario avec le champ organization.id défini

Output Ports

Output: Métadonnée sur laquelle on a ajouté org_contact.fr et org_contact.en

Paramètre

CKAN_ROOT: Root de la province.

Erreurs:

  • ERREUR: Impossible d'effectuer la requête à CKAN pour avoir les organisations --> Vérifier l'URL
  • ERREUR: Impossible d'applatir le JSON pour extraire le contenu --> Ne devrait pas arriver
  • ERREUR: Impossible d'exploser la liste result --> Vérifier le modèle de données
