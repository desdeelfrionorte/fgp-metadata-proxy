Description


  • This custom transformer corrects wrongly formed email addresses. It manages two cases: firstly when the email address do not contain the character"@", the email address is replaced by the default email adress; secondly when addresses are separated by coma, the email addresses are separated and placed into independant list of contacts{}.



Input Ports


  • INPUT: The input metadata records to process.



Output Ports


  • OUTPUT: The process output metadata records with email addresses corrected.



Parameters


  • DEFAULT_CONTACT_EMAIL: Name of the attribute containing the default email address or string character representing the default email address
